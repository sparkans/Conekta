

para instalar la base de datos necesaria en el correcto funcionamiento del sistema

accesa a la carpeta install, tienes dos opciones:

1. importar el archivo sql a tu base de datos ó

2. ejecutar el archivo installer.php

3. el archivo de configuracion donde se encuentran la base de datos el usuario,
   la contraseña, etc.se encuentra en core/settings.php 



para manejar los precios
tienes 2 opciones:
 
1. modificar el archivo settings.php esto si sera un costo constante o bien

2. modificar el archivo core/enviar-ctrl.php y asignar tus propias variables


Eso es todo lo que necesitas para iniciar el sistema, disfrutalo :D
